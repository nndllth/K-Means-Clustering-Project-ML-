# K-Means-Clustering-Project-ML-
Pengelompokan Provinsi di Indonesia Menurut Pertumbuhan Ekonomi dan Faktor yang Mempengaruhinya Tahun 2019 dan 2020 dengan Metode K-Means


CLUSTER 2019.ipynb adalah code untuk K-Means clustering pada Data tahun 2019
CLUSTER 2020.ipynb adalah code untuk K-Means clustering pada Data tahun 2020
Project ML Nanda_Lailatul_H.pdf adalah file pdf dari penjelasan atau easy report hasil clustering yang diperoleh
